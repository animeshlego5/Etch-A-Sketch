# Top Etch-a-Sketch

## Overview
The Top Etch-a-Sketch is a web-based drawing application that mimics the classic Etch-a-Sketch toy. Users can create intricate designs by interacting with a grid, changing the colors of individual squares. This project leverages HTML, CSS, and JavaScript to provide a fun and interactive drawing experience.

## Features
- **Dynamic Grid Creation**: Users can input their desired grid size, allowing for customizable drawing areas.
- **Color Change on Hover**: Each square on the grid changes color when hovered over, providing an engaging user experience.
- **Reset Functionality**: Users can easily reset the grid to start anew, maintaining a seamless drawing experience.

## Technologies Used
- **HTML**: Provides the structure for the application.
- **CSS**: Handles the styling and visual layout of the grid and buttons.
- **JavaScript**: Implements the core functionality and interactivity of the application.
